Integration tests for DatabaseImpl. There are written as integration tests as officially recommend because:

  • Different versions of android are shipped with different versions of SQLite. Integration tests allow for better coverage on them.

  • Robolectric's SQLite implementation does not match Android, e.g.: https://github.com/robolectric/robolectric/blob/master/shadows/framework/src/main/java/org/robolectric/shadows/ShadowSQLiteConnection.java#L97
